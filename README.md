# csompiler

Das ist my hobby C compiler project, where I learn about the compilation process, abstract syntax trees, intermediate representations, tokenization, grammars and other compiler related terms.

This is a C++ implementation of the book  "Nora Sandler - Writing a C Compiler". It supports all the mentioned features and data types of the book (including extra credit parts), and satisfy all the provided (1600+) test cases.

## Only for Mac OS
```
arch -x86_64 zsh
```

## Build and run the project
```
make
./build/csompiler
```
